Instalación
===========

Pre-requisitos
--------------
* Fedora 25
* git
* poppler-utils
* wget

Instalación
-----------
.. code:: bash

    dnf -y install git poppler-utils wget

Uso
---
Lee los scripts:

ley2txt:
    Transforma los PDF que descarga del sitio en texto. Te los pone en `$HOME/src/LeyesMexicanas/federales`.

ley2github:
    Sube todo a github si forkeaste el repositorio y lo clonaste.
